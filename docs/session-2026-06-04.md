## 3. Non-determinism: found + fixed (the big one)
- At temp 0 the pipeline changed its primary driver on **84%** of cases
  between two identical runs → single-run A/B is pure noise.
- Root cause: **agent 4 (the aggregator) is hardcoded `temperature = 0.2`** in
  cell 9. Made it configurable (`regen.load_pipeline(agent4_temp=…)`, wired to
  `--temperature`). Pinning it to 0 cut churn **84% → 23%**.
- Residual 23% is gemini-2.5-pro's own temp-0 variance → needs N-run averaging.
- **Production recommendation:** drop agent-4 temp to 0 in the live notebook
  for a more stable RCA (Zaidul/Pooja decision; not pushed).

## 4. N-run measurement with CIs (`evaluator/ab_measure.py`)

  accuracy as **mean ± 95% CI**. A/B = run on NEW notebook, then `pull_notebook`
  the OLD one and run again; effect is real only if CIs separate.
- Validated small-scale (FIXSMOKE 20×3 → 20/25/40%, which is *why* CIs matter).
- Fixed a single-event-loop bug (repeated `asyncio.run()` nan'd runs 2+).

## 7. Two infra fixes that unblocked clean measurement
- **Bot/matcher self-saturation (root cause of the "quota" nans).** The regen
  **bot** and the **matcher** both hit the *same* regional us-central1 2.5-pro
  per-minute bucket and saturated it at ~30 cases × 2 frameworks. **Fix:** added
  `ab_measure.py --force-global` + env-tunable `REGEN_CONCURRENCY` (default 2) →
  routes the **bot** (even the default 2.5-pro) to the **global** endpoint
  (separate quota pool) while the matcher keeps the regional pool. Decoupled,
  runs clean at concurrency 6–8. **No QPM bump needed.**
- **Global-endpoint auth bug.** The static one-shot OAuth token intermittently
  **401s `ACCESS_TOKEN_TYPE_UNSUPPORTED` under concurrency** on the global
  v1beta1 endpoint — silently decimated the **agent-4 aggregator** → nan runs
  (this was part of what looked like "quota"). **Fix:** `_install_global_genai`
  now builds **auto-refreshing ADC creds** (`google.auth.default(...).with_
  quota_project(...)`), falling back to the static token. Burst test:
  static→failures, ADC→12/12 clean. After the fix: **0 agent-4 failures**.

## 9. Matcher calibration (Reopen, 45 cases) — PASSES
- Hand-read all 45 verdicts (`evaluator/_calib_reopen.py`, dump

  bar). The 8 `Correct` are genuine exact matches; the 37 `Incorrect` are real
