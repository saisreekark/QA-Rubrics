## Decisions / clarifications

- **Accuracy metric** is **case-level**: `Correct Cases ÷ Total Audited`.
  The **primary driver** is the main check; the **secondary driver** is a
  tiebreaker / supporting signal.
- **Pass / Fail threshold**: **score ≥ 90 ⇒ Pass**, **< 90 ⇒ Fail**. Today
  only `Quality_RCA_Output` contributes to the score — by design.

  Context Caching** to preload static SOP / rule content. Goal: cut tokens,
  speed up runtime, prevent hallucinations. This is the substitute for a
  frozen regression set.
